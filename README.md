# 🎯 Soletra - Jogo de Palavras

Um jogo de formar palavras inspirado no Soletra do G1, desenvolvido com HTML, CSS e JavaScript puro.

## 📋 Sobre o Jogo

Soletra é um jogo de palavras desafiador onde você precisa formar palavras usando um conjunto de letras disponíveis. O objetivo é encontrar o máximo de palavras possível e acumular pontos!

## 🎮 Como Jogar

1. **Letra Central**: A letra amarela no centro DEVE estar presente em todas as palavras
2. **Letras Externas**: Use as 6 letras roxas ao redor para formar palavras
3. **Regras**:
   - Palavras devem ter no mínimo 4 letras
   - Cada letra pode ser usada múltiplas vezes
   - Palavras maiores valem mais pontos
   - A letra central é obrigatória em todas as palavras

## ⚡ Funcionalidades

- ✨ Interface visual moderna e responsiva
- 🔄 Botão para embaralhar as letras
- 💡 Sistema de dicas (custa 50 pontos)
- 📊 Pontuação e sistema de níveis
- 📝 Lista de palavras já encontradas
- 🎨 Animações suaves e feedback visual

## 🚀 Como Executar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Comece a jogar!

Não é necessário instalar nenhuma dependência ou servidor. O jogo roda completamente no navegador!

## 📊 Sistema de Pontuação

- 4 letras: 1 ponto
- 5 letras: 3 pontos
- 6 letras: 5 pontos
- 7 letras: 7 pontos
- 8+ letras: 10+ pontos

## 🎯 Objetivo

Encontre o máximo de palavras possível e alcance a maior pontuação! A cada 10 palavras encontradas, você avança de nível.
